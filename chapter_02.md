–û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –¥–≤–æ—Ö –∑–Ω–∞—á–µ–Ω—å.
–¶—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø–æ–≤–µ—Ä—Ç–∞—é—Ç—å –±—É–ª–µ–≤–µ –∑–Ω–∞—á–µ–Ω–Ω—è (—ñ—Å—Ç–∏–Ω—É –∞–±–æ –±—Ä–µ—Ö–Ω—é), –∑–∞—Å–Ω–æ–≤–∞–Ω–µ –Ω–∞ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è.
–û—Å—å –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –≤ Go:

- **==**: –†—ñ–≤–Ω–µ `fmt.Println(7 == 5) // –í–∏–≤–µ–¥–µ–Ω–Ω—è: false`
- **!=**: –ù–µ —Ä—ñ–≤–Ω–µ `fmt.Println(7 != 5) // –í–∏–≤–µ–¥–µ–Ω–Ω—è: true`
- **<**: –ú–µ–Ω—à–µ `fmt.Println(7 < 5) // –í–∏–≤–µ–¥–µ–Ω–Ω—è: false`

#### 2.2.3 –õ–æ–≥—ñ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏

–õ–æ–≥—ñ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –∫–æ–º–±—ñ–Ω—É–≤–∞–Ω–Ω—è –±—É–ª–µ–≤–∏—Ö –∑–Ω–∞—á–µ–Ω—å –≤ —É–º–æ–≤–Ω–∏—Ö –≤–∏—Ä–∞–∑–∞—Ö.
–û—Å—å –ª–æ–≥—ñ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –≤ Go:

- **&&**`**: –õ–æ–≥—ñ—á–Ω–µ –Ü (–ø–æ–≤–µ—Ä—Ç–∞—î true, —è–∫—â–æ –æ–±–∏–¥–≤–∞ –æ–ø–µ—Ä–∞–Ω–¥–∏ —î true) `fmt.Println(true && true) // –í–∏–≤—ñ–¥: true`
- **||**: –õ–æ–≥—ñ—á–Ω–µ –ê–ë–û (–ø–æ–≤–µ—Ä—Ç–∞—î true, —è–∫—â–æ —Ö–æ—á–∞ –± –æ–¥–∏–Ω –∑ –æ–ø–µ—Ä–∞–Ω–¥—ñ–≤ —î true) `fmt.Println(true || false) // –í–∏–≤—ñ–¥: true`
- **!**: –õ–æ–≥—ñ—á–Ω–µ –ù–ï (–ø–æ–≤–µ—Ä—Ç–∞—î true, —è–∫—â–æ –æ–ø–µ—Ä–∞–Ω–¥ —î false, —Ç–∞ false, —è–∫—â–æ –æ–ø–µ—Ä–∞–Ω–¥ —î true)

#### 2.2.4 –ë—ñ—Ç–æ–≤—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏

–ë—ñ—Ç–æ–≤—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –≤–∏–∫–æ–Ω–∞–Ω–Ω—è –æ–ø–µ—Ä–∞—Ü—ñ–π –Ω–∞–¥ –¥–≤—ñ–π–∫–æ–≤–∏–º –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è–º —á–∏—Å–µ–ª.
–û—Å—å –±—ñ—Ç–æ–≤—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –≤ Go:

- **&**: –ë—ñ—Ç–æ–≤–∏–π AND `bitwiseAnd := 7 & 5 // bitwiseAnd —î 5 (binary 0111 & 0101 = 0101)`

#### 2.2.5 –û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è

–û–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è –∑–Ω–∞—á–µ–Ω—å –∑–º—ñ–Ω–Ω–∏–º.
Go –º–∞—î –æ—Å–Ω–æ–≤–Ω–∏–π –æ–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è `=` —Ç–∞ —Å–∫–ª–∞–¥–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è,
—â–æ –≤–∏–∫–æ–Ω—É—é—Ç—å –æ–ø–µ—Ä–∞—Ü—ñ—é —Ç–∞ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è –∑–∞ –æ–¥–∏–Ω –∫—Ä–æ–∫.
–û—Å—å –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è –≤ Go:

- `:=`: –î–µ–∫–ª–∞—Ä–∞—Ü—ñ—è —Ç–∞ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è `x:= 5 //–¥–µ–∫–ª–∞—Ä—É—î–º–æ —Ç–∞ –ø—Ä–∏—Å–≤–æ—é—î–º–æ x –∑–Ω–∞—á–µ–Ω–Ω—è 5`
- `=`: –ü—Ä–∏—Å–≤–æ—î–Ω–Ω—è

  x = 5 // x —Ç–µ–ø–µ—Ä 5
  ```
- `+=`: –î–æ–¥–∞—Ç–∏ —Ç–∞ –ø—Ä–∏—Å–≤–æ—ó—Ç–∏

  ```

  ```
- `*=`: –ü–æ–º–Ω–æ–∂–∏—Ç–∏ —Ç–∞ –ø—Ä–∏—Å–≤–æ—ó—Ç–∏

  ```

  ```
- `%=`: –ú–æ–¥—É–ª—å —Ç–∞ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è

  ```

  ```
- `|=`: –ë—ñ—Ç–æ–≤–∞ `OR` –æ–ø–µ—Ä–∞—Ç–æ—Ä —Ç–∞ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è

  ```

  ```

  ```

- `&^=`: –û—á–∏—Å—Ç–∏—Ç–∏ –±—ñ—Ç —Ç–∞ –ø—Ä–∏—Å–≤–æ—ó—Ç–∏

–¢–µ–ø–µ—Ä —É –≤–∞—Å —î –ø—Ä–∏–∫–ª–∞–¥–∏ –¥–ª—è –≤—Å—ñ—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä—ñ–≤ Go, —è–∫—ñ –ø–æ–≤–∏–Ω–Ω—ñ –¥–æ–ø–æ–º–æ–≥—Ç–∏ –≤–∞–º –∑—Ä–æ–∑—É–º—ñ—Ç–∏, —è–∫ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –∫–æ–∂–µ–Ω –∑ –Ω–∏—Ö —É
–≤–∞—à–∏—Ö –ø—Ä–æ–≥—Ä–∞–º–∞—Ö Go.

–¶—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ —Å—Ç–∞–Ω–æ–≤–ª—è—Ç—å –æ—Å–Ω–æ–≤—É —Ä–æ–±–æ—Ç–∏ –∑ –¥–∞–Ω–∏–º–∏ —ñ –ª–æ–≥—ñ–∫–æ—é, –¥–æ–∑–≤–æ–ª—è—é—á–∏ –≤–∞–º —Å—Ç–≤–æ—Ä—é–≤–∞—Ç–∏ –±—ñ–ª—å—à —Å–∫–ª–∞–¥–Ω—ñ –¥–æ–¥–∞—Ç–∫–∏ —ñ –≤–∏—Ä—ñ—à—É–≤–∞—Ç–∏

### 2.3 –î–µ–∫–ª–∞—Ä–∞—Ü—ñ—è —Ç–∞ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—è –∑–º—ñ–Ω–Ω–∏—Ö

–ó–º—ñ–Ω–Ω—ñ —î –±—É–¥—ñ–≤–µ–ª—å–Ω–∏–º–∏ –±–ª–æ–∫–∞–º–∏ –±—É–¥—å-—è–∫–æ—ó –º–æ–≤–∏ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è, –¥–æ–∑–≤–æ–ª—è—é—á–∏ –≤–∞–º –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ —Ç–∞ –º–∞–Ω—ñ–ø—É–ª—é–≤–∞—Ç–∏ –¥–∞–Ω–∏–º–∏.

–£ —Ü—å–æ–º—É —Ä–æ–∑–¥—ñ–ª—ñ –º–∏ —Ä–æ–∑–≥–ª—è–Ω–µ–º–æ –Ω–∞–π–ø–æ—à–∏—Ä–µ–Ω—ñ—à—ñ –º–µ—Ç–æ–¥–∏ –¥–µ–∫–ª–∞—Ä–∞—Ü—ñ—ó —Ç–∞ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—ó –∑–º—ñ–Ω–Ω–∏—Ö –≤ Go.

#### 2.3.1 –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –∫–ª—é—á–æ–≤–æ–≥–æ —Å–ª–æ–≤–∞ var

#### 2.3.2 –í–∏–∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—É –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é :=

–ö–æ–º–ø—ñ–ª—è—Ç–æ—Ä –∞–≤—Ç–æ–º–∞—Ç–∏—á–Ω–æ –≤–∏–≤–µ–¥–µ —Ç–∏–ø –Ω–∞ –æ—Å–Ω–æ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è.
–©–æ–± –æ–≥–æ–ª–æ—Å–∏—Ç–∏ —Ç–∞ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –∑–º—ñ–Ω–Ω—É –∑ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è–º –≤–∏–≤–µ–¥–µ–Ω–Ω—è —Ç–∏–ø—É, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä `:=`:
